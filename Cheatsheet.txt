MY GITHUB CHEAT SHEET

1. START NEW PROBLEM:
   nano Name.cpp
   (Paste code, Save: Ctrl+O, Exit: Ctrl+X)

2. UPLOAD CODE (Feature):
   git add .
   git commit -m "feat: solved [Problem Name]"
   git push

3. UPDATE LIST (Docs):
   nano README.md
   (Copy last row, paste, edit name and link)
   
   git add .
   git commit -m "docs: added [Problem Name] to list"
   git push
